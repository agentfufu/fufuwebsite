What is the lightning network ?

C‘est quoi le lightning network ?

Qué es el lightning network ?




The Lightning Network is simply a second layer to the Bitcoin network allowing fast, low-fee, peer-to-peer transactions through a Lightning channel.

Le Lightning Network est simplement une deuxiéme couche du réseau Bitcoin permettant des transactions peer-to-peer rapides et peu coûteuses via un canal Lightning.

Lightning Network es simplemente una segunda capa de la red Bitcoin que permite transacciones de persona a persona rápidas y de bajo costo a través de un canal Lightning.


Lightning Network Features

Caractéristiques de Lightning Network

Caracteristicas del Lightning Network


  • Flexibility : Units smaller than the base layer of bitcoin can be traded.

  • Flexibilité : Des montants plus petits que la base du bitcoin peuvent être échangées.

  • Flexibilidad : se pueden intercambiar unidades más pequeñas que la capa base de bitcoin.

  • Speed : Payments can be settled much faster than on the blockchain.

  • Rapidité : les paiements peuvent être réglés beaucoup plus rapidement que sur la blockchain.

  • Velocidad : los pagos se pueden hacer mucho más rápido que en la blockchain.

  • Privacy : Individual payment details aren't publicly recorded on the blockchain.

  • Confidentialité : les détails de paiement individuels ne sont pas enregistrés publiquement sur la blockchain.

  • Privacidad : los detalles de pago individuales no se registran públicamente en la blockchain.
